Markov Chain (MC)

Consider the variable y(t) is the vector (state) at time t. The Markov chain has the following form:

(1)y(t)=Ay(t−1)

where A is the transition matrix.

Hidden Markov Model (HMM)

Similar to Markov chain. The Hidden Markov model has an additional hidden vector (state):

(2)h(t)=Ah(t−1)y(t)=By(t−1)

where ht is the "hidden" vector (state) at time t.

Recurrent Neural Network (RNN)

(3)h(t)=σ(Wx(t)+Uh(t−1)+b)o(t)=σ(Vh(t)+b)

where xi is the input, oi is the output and σ is the activation function.

Long Short-Term Memory (LSTM)

LSTM is the modified version of RNN, which additionally has memory cell state ct.

(4)c(t)=f(t)⋅c(t−1)+i(t)⋅c~(t)

where ft is the forget gate, it is the input gate and ct~ is the cell input.

(5)f(t)=σ(Wfx(t)+Ufh(t−1)+bf)i(t)=σ(Wix(t)+Uih(t−1)+bi)c~(t)=σ(Wcx(t)+Uch(t−1)+bc)

The hidden state h(t) and output o(t) are defined as

(6)h(t)=ot⋅σ(c(t))o(t)=σ(Wox(t)+Uoh(t−1)+bo)
