Self-Attention

Self-attention is a kind of layer / mechanism which is widely used in NLP. It contains three kinds of components:

(1)qi=Wqxiki=Wkxivi=Wvxi

where qi is query, ki is key and vi is value. The output oi is designed as:

(2)aj,i=softmax{kjqidk}oi=∑jvjaj,i

where aj,i is the "attention matrix", which means the relation between vj and oi. This layer also can be written as matrix form so that the parallel computing is possible.

(3)Q=WqxK=WkxV=WvxA=softmax{KTQdk}o=VA

For the multi-head self-attention, it has multiple query (Qα) / key (Kα) / value (Vα) and thus it has multiple outputs Oα. The final output is the linear combination of multiple outputs:
